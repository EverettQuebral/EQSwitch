#### Description
This is an open source project for controlling an ASCOM Switch controlled by Arduino and a 4 relay arduino Module

##### Hardware Parts
- 1 Arduino UNO
- 1 4 Relay Module
- 4 Pull Up Switch
- 4 220 Ohms Resistors
- 1 12v to 5v DC Step down transformer
- 1 On/Off Switch
- 1 120v to 12v ACDC Converter
